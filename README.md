# Solutions

A personal space for my competitive programming solutions.

Feel free to browse it, although the content might not be very well organized or correct.

## Status

> **This repository is unmaintained.**

This was an active competitive programming scratchpad several years ago (last meaningful commit dates to 2017).
It is kept here for historical reference only: no further updates or bug fixes are planned, and pull requests will generally not be reviewed.

The code is not intended for production use. Use it at your own discretion.

## Contents

Solutions are grouped into one directory per online judge host, named after the host the solutions were originally submitted to (these directory names reflect the URLs used at the time and are kept unchanged for historical traceability):

- `uva.onlinejudge.org/`
- `spoj.com/`
- `codeforces.com/`
- `coj.uci.cu/`
- `icpcarchive.ecs.baylor.edu/`
